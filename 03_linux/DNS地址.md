## 六、配置 DNS 的方法
1. **自动获取**：大多数情况下，路由器或设备会自动分配 DNS 服务器地址。
2. **手动设置**：
   - 在设备的网络设置中，找到 TCP/IPv4 或 TCP/IPv6 配置。
   - 选择“使用下面的 DNS 服务器地址”，输入主 DNS 和备用 DNS 地址。
   - 保存并重启网络连接。
